Creating a snake game

A simple snake style game

This simplified version of Snake focuses on logic, making sure that the snake doesn't move off of the grid, increases speed and grows when it 'eats' an apple and so on...

There are 19 different JavaScript methods used in the Snake game, some of the methods included are:

  • addEventListener()
  • .pop()
  • .unshift()
  • .push()
  • setTimeout()
  • setInterval()
  • .keyCode()

The styling throughout he course tutorial was very basic, I have since gone in to add my own original styling to the game using CSS.
