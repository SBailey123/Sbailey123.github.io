Creating a blackjack game

A simple blackjack style game

Blackjack style game made using HTML, CSS and JavaScript

This was one of the first JavaScript projects within the Scrimba course.

Like most projects it started with the HTML mark-up creating elements, classes and id's that were needed. Next came the JavaScript, where we needed to implement all of the JavaScript methods that had been taught in the previous lessons.
The JavaScript behind the game is made up of:

  • if/else statements
  • boolean conditions
  • For loops
  • Returning values
  • Math.floor() and Math.randon()
  • Logic operators
