Creating a game of war card game

Game of war

This is the classic card game, game of war. Simply put, highest card wins.

This project was created using JavaScript and introduced the concept of using API's. The API used was a deck of cards which included the 52 playing cards and their images.

The game itself relies heavily on if/else statements in order to determine which card is higher and who gets the point.
