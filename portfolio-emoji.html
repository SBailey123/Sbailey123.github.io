Creating an emoji list

creating an emoji list

This project enabled understanding of list creation.

It focuses heavily on the .push(), .pop(), .shift() and .unshift() allowing us to add and remove items from the beginning and end of the array.

Whilst this project was laid out as an emoji list, it gives the fundementals for creating a list application. Whether it may be a shopping list or to do list. it allows you to add or remove items from the list.
